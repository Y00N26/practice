## 1. 지금까지 한 것 (완료 ✅)

- `create-next-app` 으로 `lunch-survey` 프로젝트 생성
- Next.js `app` 라우터 + Tailwind 기본 세팅 유지
- 페이지 구조 만들기
  - `/` → 서비스 소개 화면
    - 서비스 설명 문구
    - `설문 시작하기` 버튼 (`/survey` 로 이동)
  - `/survey` → 한 질문씩 나오는 **step 형식 설문**
    - Q1. 식사 인원 (혼밥 / 2인 / 4인이상 / 10인 이상 단체)
    - Q2. 비용 (1만원 이하 / 1만원 초중반 / 1만원 후반 / 2만원)
    - Q3. 분위기 (캐주얼 / 분위기 / 정갈함)
    - Q4. 거리 (5분 이내 / 10분 이내 / 15분 이내)
  - `/result` → 추천 결과 화면
    - TOP3 식당 카드로 표시
    - `1위 2위 3위` 숫자 출력
    - 태그: people / price / mood / distance
- 추천 로직 구현 (`src/lib/recommend.ts`)
  - 주어진 10개 식당 JSON 하드코딩
  - 1단계: **완전 일치 필터**
  - 2단계: 없으면 **가중치 점수 계산**
    - people: 40, price: 30, distance: 20, mood: 10
    - 거리 라벨 ↔ 숫자(5, 10, 15) 변환
- 모바일 우선 UI
  - 카드형 레이아웃
  - 큰 버튼
  - 그라디언트 배경, 중앙 카드
